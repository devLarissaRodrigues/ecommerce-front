# StoreFront

## MAKE SURE YOU HAVE [ADMIN](https://github.com/luizaugustoliveira/AdminDashboard-ProjetoProgWeb) SETUP FIRST!

### Prerequisites

**Node version 14.x**

### Cloning the repository

```shell
git clone https://github.com/luizaugustoliveira/ProjetoProgWebStore.git
```

### Install packages

```shell
npm i
```

### Setup .env file

```js
NEXT_PUBLIC_API_URL=
```

### Start the app

```shell
npm run dev
```
